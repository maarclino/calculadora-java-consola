Calculadora en Java por consola

Este proyecto es una calculadora simple que permite realizar operaciones básicas (suma, resta, multiplicación y división) entre dos números mediante un menú interactivo en consola. El programa solicita dos números, realiza la operación seleccionada y muestra el resultado con dos decimales. Incluye control para evitar la división entre cero y muestra un mensaje en caso de seleccionar opciones no válidas.

Requisitos:

JDK 8 o superior.
Un editor de código.
Cómo compilar: javac Calculadora.java

Cómo ejecutar: java Calculadora
